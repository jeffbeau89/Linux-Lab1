# Enterprise Linux Automation Lab

## Project Overview
A multi-server Linux infrastructure deployed on AWS and hardened to 
enterprise security standards. Built to demonstrate real-world sysadmin 
skills including OS hardening, service configuration, centralized logging, 
and infrastructure automation.
